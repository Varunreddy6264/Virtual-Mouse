Virtual Mouse Using Hand Gestures

Control your computer's mouse using only hand gestures detected through a webcam. This system uses computer vision and machine learning to track hand landmarks in real-time and translate them into mouse actions such as movement, click, double-click, right-click, and scroll. It provides a touchless and intuitive way to interact with your device, enhancing accessibility and user experience.
