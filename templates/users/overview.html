Virtual Mouse Using Hand Gestures

📌 Project Summary

This project is a contactless virtual mouse system that allows users to control their computer cursor using hand gestures. Leveraging real-time hand tracking with a webcam and computer vision, the system interprets specific finger positions to perform mouse functions like movement, clicks, and scrolling. It aims to offer an intuitive and hygienic way of interacting with computers.

✋ Recognized Hand Gestures

📌 Index Finger Up – Move Cursor
📌 Index + Thumb Touch – Left Click
📌 Middle + Thumb Touch – Right Click
📌 Index + Thumb Double Touch – Double Click
📌 Index + Thumb hold 1 sec – drag and drop
📌 Fast swipe left/right – Switch Tabs
📌 Thumb–Index distance – Volume Control
📌 All 5 fingers open ✋ – Screenshot
📌 fingers should move up and down ✋ – Scrolling

🔧 Technologies Used

  • Python 3.11.2
  • OpenCV – For image capture and processing
  • MediaPipe – For real-time hand tracking
  • PyAutoGUI – For GUI automation like mouse events
  • Django – For web integration
  • HTML/CSS – For frontend design

📚 Libraries Used

  • cv2 (OpenCV)
  • mediapipe
  • pyautogui
  • numpy
  • time

🧠 Algorithms & Concepts

  • Hand Landmark Detection using MediaPipe
  • Euclidean Distance Calculation between fingers
  • Gesture Recognition based on finger positions
  • Coordinate mapping from camera frame to screen resolution
  • Thresholding to reduce false positives

✨ Features Implemented

  • Move mouse cursor using index finger
  • Single click with index and thumb touching
  • Right click with middle finger and thumb touching
  • Double click with index, middle, and thumb close together
  • Scrolling by moving two fingers vertically
  • Fail-safe feature to stop operation when hand leaves frame
